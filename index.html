Set up Telegram

Follow these steps to create your bot

1

Open Telegram and search for @BotFather

BotFather is Telegram's official bot creator. Tap the one with the blue checkmark.

2

Send /newbot

Type /newbot and send it. BotFather will ask for a name.

3

Choose a name and username

Name it "Your Bot". Then pick a username ending in "bot" (like yourbot_bot).

4

Copy the token

BotFather gives you a token like 123456789:ABCdef.... Copy it.

1

Go to Discord Developer Portal

Open discord.com/developers/applications

2

Create New Application

Click "New Application", name it "Your Bot"

3

Add Bot and enable intents

Go to Bot tab, click "Add Bot". Enable "Message Content Intent" under Privileged Gateway Intents.

4

Copy the token

Click "Reset Token" and copy it.

5

Invite to your server

Go to OAuth2 > URL Generator. Check "bot" and "applications.commands". Copy URL and open it.

Go to Slack API

Open api.slack.com/apps, click "Create New App"

3

Add permissions

Go to OAuth & Permissions, add scopes: chat:write, im:history, im:read, im:write, users:read

Run your bot

Follow these steps exactly

Checklist

1
Extract the zip
Double-click FreeClaw.zip to extract it to a folder
2
Open the folder
You'll see setup.sh, config.yaml, and README.txt
3
Run the setup script
Mac/Linux: Open Terminal, drag the setup.sh file into it, press Enter
Windows: Double-click setup.bat
4
Wait 2-3 minutes
The script installs everything automatically. You'll see "Bot is running" when it's ready.
5
Test your bot
Open Telegram and send a message to your bot!
Remember: Your bot runs on your computer. When your computer is off or asleep, the bot is offline. When you turn it back on, just run the setup script again.
