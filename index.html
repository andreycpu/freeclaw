Set up Telegram

Follow these steps to connect your assistant. Don't worry, we'll walk you through it.

1

Open Telegram and search for @BotFather

BotFather is Telegram's official bot for creating other bots. Just search "BotFather" and tap the verified one with the checkmark.

2

Send /newbot to BotFather

Type /newbot and send it. BotFather will ask you for a name.

3

Choose a name and username

First, give it a display name (like "Your Assistant"). Then pick a username ending in "bot" (like "your_assistant_bot").

4

Copy the token

BotFather will give you a token that looks like 123456789:ABCdefGHI.... Copy it and paste it below.

1

We'll show you a QR code

After you download and run the app, it will show a QR code on screen.

2

Open WhatsApp on your phone

Go to Settings > Linked Devices > Link a Device

3

Scan the QR code

Point your phone at the QR code. That's it - you're connected!

1

Go to Discord Developer Portal

Open discord.com/developers/applications and log in.

2

Create a new application

Click "New Application" and give it a name (like "Your Assistant").

3

Add a bot

Go to "Bot" in the sidebar, click "Add Bot". Turn on "Message Content Intent" under Privileged Gateway Intents.

4

Copy the token

Click "Reset Token" and copy it. Paste it below.

5

Invite to your server

Go to OAuth2 > URL Generator. Check "bot" and "applications.commands". Copy the URL and open it to add the bot to your server.

1

Go to Slack API

Open api.slack.com/apps and click "Create New App".

2

Create from scratch

Choose "From scratch", name it "Your Assistant", and pick your workspace.

3

Add bot permissions

Go to "OAuth & Permissions", scroll to "Bot Token Scopes" and add: chat:write, im:history, im:read, im:write, users:read

4

Install to workspace

Click "Install to Workspace" and authorize.

5

Copy the Bot Token

Copy the "Bot User OAuth Token" (starts with xoxb-) and paste it below.

What does "local" mean?

Your assistant runs on YOUR computer, not in the cloud. When your computer is on, your assistant is awake. When you shut down, it goes to sleep. When you turn it back on, it wakes up with all its memories intact. Your data never leaves your machine.

Local vs Server

Feature Local This Setup Server
Available 24/7 Only when computer is on Yes, always online

Your data stays private 100% on your machine Lives on a server
No ongoing costs Just your API key Server costs ~$5-20/mo
Setup difficulty 5 minutes, no coding Requires technical setup
Memory persists Yes, stored locally Yes, stored on server
