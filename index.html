Your own AI assistant, running locally

Run OpenClaw on your own computer. No servers, no complexity. Just install and connect to Telegram, Discord, or WhatsApp.

1 Open Terminal and run this command

Press Cmd + Space, type "Terminal", press Enter. Then paste this:

curl -fsSL https://openclaw.ai/install.sh | bash

This installs Node.js (if needed) and OpenClaw. Takes about 2 minutes.

2 Start the setup wizard

After installation, run:

openclaw init

This walks you through connecting Telegram, Discord, or WhatsApp.

3 Start your assistant

openclaw gateway start

Your AI assistant is now running. Message it on Telegram/Discord/WhatsApp from any device.
