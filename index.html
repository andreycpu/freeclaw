Set up Telegram

Follow these steps exactly

1

Open Telegram, search @BotFather

It's the official bot with a blue checkmark

2

Send /newbot

BotFather will ask for a name

3

Name it "YourBot"

Then pick a username ending in "bot"

4

Copy the token

Looks like 123456789:ABCdef...

1

Go to Discord Developer Portal

2

New Application → name it "YourBot"

3

Bot tab → Add Bot → Enable Message Content Intent

4

Reset Token and copy it

5

OAuth2 → URL Generator → check "bot" + "applications.commands"

Copy URL, open it, add to your server

1

No token needed for WhatsApp

You'll scan a QR code after running the bot

2

When you run the setup...

Open WhatsApp → Settings → Linked Devices → scan the QR

1

Go to api.slack.com/apps

2

Create New App → From scratch

Name: "YourBot", pick workspace

3

OAuth → Bot Token Scopes

Add: chat:write, im:history, im:read, im:write, users:read

4

Install to Workspace → Copy Bot Token

Starts with xoxb-

This stays on your computer. Never sent to us.

Get this from your AI provider's dashboard

Having trouble?
BotFather not responding? Make sure you're messaging @BotFather (with the blue checkmark), not a fake.
Token looks wrong? Telegram tokens have a colon in the middle. Discord tokens are one long string.

Run your bot

Follow along - each step matches what you'll see

Installation Checklist

1
Extract the zip
Double-click FreeClaw.zip to extract
2
Open the folder
You'll see setup.sh, config.yaml, README.txt
3
Run the setup
Mac: Open Terminal, drag setup.sh into it, press Enter
Windows: Double-click setup.bat
4
Wait for "Bot is running"
Takes 2-3 minutes. You'll see progress in the terminal.
5
Test it!
Open Telegram and message your bot
Remember: Your bot runs on your computer. You chat with it from anywhere (including your phone). When your computer is off, the bot sleeps. Turn it on and run the script again to wake it up.
Having trouble?
"Permission denied" on Mac? Run: chmod +x setup.sh then try again
Script closes immediately on Windows? Open Command Prompt first, then drag the .bat file into it
Stuck on "Installing..."? This is normal for first run. Wait up to 5 minutes.
