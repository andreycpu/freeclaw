# FreeClaw

Your own AI chatbot in 2 minutes. No coding required.

## How it works

1. Visit the website and pick your AI provider + chat platform
2. Download the setup script
3. Double-click the file - your bot starts automatically

## What gets installed

- [OpenClaw](https://openclaw.ai/) - open-source personal AI assistant platform
- Portable Node.js runtime (doesn't affect your system)
- Your bot config and personality files

Everything runs on your computer. Your API keys never leave your machine.

## Supported

- **AI Providers:** Claude (Anthropic), ChatGPT (OpenAI), Gemini (Google)
- **Chat Platforms:** Telegram, Discord, WhatsApp, Slack
- **Operating Systems:** Windows 10+, macOS 12+
