Deploy OpenClaw on Telegram

Complete step by step guide. No experience needed.

About 15 minutes Cost: $4/month or free
Before you start

What you will need

  • A computer with internet access (Mac, Windows, or Linux)
  • A Telegram account (the app you already use)
  • An Anthropic or OpenAI API key (I will show you how to get one)
  • 15 minutes of uninterrupted time
Step 1

Get a server

A server is a computer that runs 24/7 so your AI assistant is always available. You will rent one from a hosting company.

DigitalOcean Recommended

Costs $4 per month. Simplest option for beginners.

  1. Go to digitalocean.com
  2. Click Sign Up and create an account (you will need a credit card)
  3. Once logged in, click the green Create button at the top right
  4. Select Droplets
  5. Under "Choose an image", make sure Ubuntu is selected and the version says 22.04
  6. Under "Choose Size", click Basic, then select the $4/mo option (1 GB RAM)
  7. Under "Choose a datacenter region", pick the one closest to you
  8. Under "Authentication", select Password
  9. Enter a strong password and write it down somewhere safe
  10. Click Create Droplet at the bottom
  11. Wait about 30 seconds. You will see your new server with an IP address like 123.45.67.89
  12. Copy this IP address. You will need it in the next step.

Step 2

Connect to your server

Now you will connect to your server using a tool called SSH. This lets you type commands on the server from your computer.

If you use Mac or Linux
  1. Open the Terminal app (on Mac: press Cmd+Space, type "Terminal", press Enter)
  2. Type this command, but replace YOUR_IP with the IP address from Step 1:
Terminal
ssh root@YOUR_IP
  1. It will ask "Are you sure you want to continue connecting?" Type yes and press Enter
  2. Enter the password you created in Step 1
  3. You will not see the password as you type. This is normal. Just type it and press Enter.
If you use Windows
  1. Download PuTTY (click the link that says "64-bit x86")
  2. Open PuTTY
  3. In the "Host Name" field, enter your IP address from Step 1
  4. Click Open
  5. If you see a security alert, click Accept
  6. When it asks "login as:", type root and press Enter
  7. Enter your password from Step 1

Step 3

Install OpenClaw

Now you will install the software. Copy each command, paste it into your terminal, and press Enter. Wait for each one to finish before running the next.

First, update the server software:

This takes about 1 minute
sudo apt update && sudo apt upgrade -y

Install Node.js (the programming language OpenClaw uses):

This takes about 30 seconds
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

Verify Node.js installed correctly:

Should show a version number like v20.x.x
node --version

Install OpenClaw:

This takes about 1 minute
sudo npm install -g openclaw

Create a folder for your assistant and go into it:

mkdir ~/assistant && cd ~/assistant

Initialize OpenClaw:

This creates the config file
openclaw init

Step 4

Create your Telegram bot

Telegram has a special bot called BotFather that creates new bots for you.

  1. Open Telegram on your phone or computer
  2. Search for @BotFather (make sure it has a blue checkmark)
  3. Start a chat with BotFather and send: /newbot
  4. BotFather will ask for a name. This is what people see. Example: My AI Assistant
  5. BotFather will ask for a username. This must end in "bot". Example: myai_helper_bot
  6. BotFather will reply with a message containing your token. It looks like this:
    1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
  7. Copy this token and save it somewhere safe. You will need it in the next step.
ðŸ”’

Keep your token secret. Anyone who has this token can control your bot. Do not share it publicly.

Now get your Telegram user ID:

  1. In Telegram, search for @userinfobot
  2. Start a chat and send any message
  3. It will reply with your user ID, a number like 123456789
  4. Copy this number. This tells OpenClaw that you are the owner.
Step 5

Get an AI API key

OpenClaw needs an API key to talk to an AI model. I recommend Anthropic (Claude) or OpenAI (GPT).

Anthropic (Claude) Recommended
  1. Go to console.anthropic.com
  2. Sign up for an account
  3. Go to API Keys in the left sidebar
  4. Click Create Key
  5. Copy the key. It starts with sk-ant-
  6. Add some credits to your account (minimum $5). Go to Billing in the sidebar.

Cost: Typical usage is $5 to $20 per month depending on how much you chat.

Step 6

Configure OpenClaw

Now you will edit the configuration file to add your bot token and API key.

Open the config file:

This opens a text editor
nano openclaw.yaml

Delete everything in the file and paste this:

Replace the placeholder values with your actual values
llm:
  provider: anthropic
  apiKey: YOUR_ANTHROPIC_API_KEY
  model: claude-sonnet-4-20250514

channels:
  telegram:
    token: YOUR_TELEGRAM_BOT_TOKEN
    ownerIds:
      - YOUR_TELEGRAM_USER_ID

Replace these values:

  • YOUR_ANTHROPIC_API_KEY with your API key from Step 5
  • YOUR_TELEGRAM_BOT_TOKEN with your bot token from Step 4
  • YOUR_TELEGRAM_USER_ID with your user ID from Step 4

Save the file:

  1. Press Ctrl + X (or Cmd + X on Mac)
  2. Press Y to confirm you want to save
  3. Press Enter to confirm the filename
Step 7

Start your assistant

This is the final step. You will start OpenClaw and test that it works.

This starts OpenClaw
openclaw gateway start

You should see output that says OpenClaw is running.

Now test it:

  1. Open Telegram
  2. Search for your bot by its username (the one ending in "bot")
  3. Send it a message like "Hello!"
  4. It should reply within a few seconds
