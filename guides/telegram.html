Deploy OpenClaw on Telegram

A complete step-by-step guide. No experience required.

‚è±Ô∏è ~15 minutes üí∞ ~$5/month (or free tier) üìä Beginner friendly

What we'll do:

  1. Get a server (5 min)
  2. Connect to your server (3 min)
  3. Install OpenClaw (3 min)
  4. Create your Telegram bot (2 min)
  5. Configure and launch (2 min)
1

Get a Server

You need a small server to run OpenClaw 24/7. Here are your options:

Option A: Railway (Easiest - has free tier)

  1. Go to railway.app
  2. Sign up with GitHub
  3. Click "New Project" ‚Üí "Empty Project"
  4. Skip to Step 3 (Railway has a different setup)

Option B: DigitalOcean ($4/month - more control)

  1. Go to digitalocean.com and create an account
  2. Click "Create" ‚Üí "Droplets"
  3. Choose Ubuntu 22.04
  4. Select the $4/month plan (1GB RAM is enough)
  5. Choose a region close to you
  6. Under "Authentication", choose Password and set a strong password
  7. Click "Create Droplet"
  8. Copy the IP address shown (looks like 123.45.67.89)

Option C: AWS Free Tier (Free for 12 months)

  1. Go to aws.amazon.com
  2. Create an account (requires credit card but won't charge)
  3. Go to EC2 ‚Üí Launch Instance
  4. Choose Ubuntu 22.04, t2.micro (free tier eligible)
  5. Create and download a key pair (.pem file)
  6. Launch and note the public IP

My recommendation: If you're new, start with DigitalOcean. It's $4/month but the simplest to use.

2

Connect to Your Server

Now we'll connect to your server using SSH (a secure way to control remote computers).

On Mac/Linux:

Open Terminal and type:

ssh root@YOUR_IP_ADDRESS

Replace YOUR_IP_ADDRESS with the IP from Step 1. Enter your password when asked.

On Windows:

  1. Download PuTTY
  2. Open PuTTY, enter your IP address
  3. Click "Open"
  4. Login as root with your password

You're in! You should see a command prompt like root@ubuntu:~#

3

Install OpenClaw

Copy and paste these commands one at a time. I'll explain what each does.

First, update your system:

# Updates the list of available software
sudo apt update && sudo apt upgrade -y

Install Node.js:

# Installs Node.js (the programming language OpenClaw uses)
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

Install OpenClaw:

# Installs OpenClaw globally
sudo npm install -g openclaw

Create a workspace:

# Creates a folder for your assistant
mkdir ~/my-assistant && cd ~/my-assistant

Initialize OpenClaw:

# Sets up the configuration
openclaw init
4

Create Your Telegram Bot

Now we need to create a bot on Telegram that your assistant will use.

  1. Open Telegram and search for @BotFather
  2. Send /newbot
  3. Choose a name for your bot (e.g., "My AI Assistant")
  4. Choose a username ending in "bot" (e.g., "myai_assistant_bot")
  5. BotFather will give you a token - copy it!

Keep your token secret! Anyone with this token can control your bot.

The token looks like this: 1234567890:ABCdefGHIjklMNOpqrsTUVwxyz

5

Configure and Launch

Almost there! Now we'll configure OpenClaw with your bot token and AI provider.

Open the config file:

nano openclaw.yaml

Add this configuration (replace the placeholders):

llm:
provider: anthropic # or openai
apiKey: YOUR_ANTHROPIC_API_KEY
model: claude-sonnet-4-20250514

channels:
telegram:
token: YOUR_TELEGRAM_BOT_TOKEN
ownerIds:
- YOUR_TELEGRAM_USER_ID

Save with Ctrl+X, then Y, then Enter.

Need an API key? Get one from Anthropic or OpenAI.

Find your Telegram user ID:

  1. Message @userinfobot on Telegram
  2. It will reply with your user ID (a number)
  3. Add that number to ownerIds

Start OpenClaw:

# Start the gateway (runs in background)
openclaw gateway start

That's it! Open Telegram and message your bot. It should respond!
