Deploy on Telegram

Complete guide. No experience needed.

~15 minutes $4/mo or free tier Beginner friendly
In this guide
  1. Get a server
  2. Connect to your server
  3. Install OpenClaw
  4. Create your Telegram bot
  5. Configure and launch
1

Get a server

You need a small server to run OpenClaw 24/7. Here are your options:

DigitalOcean — $4/month (recommended for beginners)

  1. Go to digitalocean.com and create an account
  2. Click Create → Droplets
  3. Choose Ubuntu 22.04
  4. Select the $4/month plan
  5. Choose Password authentication and set a strong password
  6. Click Create Droplet
  7. Copy the IP address (looks like 123.45.67.89)

AWS Free Tier — Free for 12 months

  1. Go to aws.amazon.com
  2. Create an account (requires credit card but won't charge)
  3. Go to EC2 → Launch Instance
  4. Choose Ubuntu 22.04, t2.micro (free tier)
  5. Create and download a key pair

My recommendation: Start with DigitalOcean. $4/month, simplest setup.

2

Connect to your server

On Mac/Linux: Open Terminal and run:

ssh root@YOUR_IP_ADDRESS

On Windows: Download PuTTY, enter your IP, click Open, login as root.

You should see a prompt like root@ubuntu:~#

3

Install OpenClaw

Copy and paste these commands one at a time:

# Update system
sudo apt update && sudo apt upgrade -y
# Install Node.js
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs
# Install OpenClaw
sudo npm install -g openclaw
# Create workspace and initialize
mkdir ~/my-assistant && cd ~/my-assistant
openclaw init
4

Create your Telegram bot

  1. Open Telegram, search for @BotFather
  2. Send /newbot
  3. Choose a name (e.g., "My AI Assistant")
  4. Choose a username ending in "bot"
  5. Copy the token BotFather gives you

Keep your token secret. Anyone with it can control your bot.

5

Configure and launch

Open the config file:

nano openclaw.yaml

Add this (replace the placeholders):

llm:
provider: anthropic
apiKey: YOUR_API_KEY
model: claude-sonnet-4-20250514

channels:
telegram:
token: YOUR_BOT_TOKEN
ownerIds:
- YOUR_USER_ID

Save with Ctrl+X, then Y, then Enter.

Get your user ID: Message @userinfobot on Telegram.

Start OpenClaw:

openclaw gateway start

Done! Message your bot on Telegram. It should respond.
